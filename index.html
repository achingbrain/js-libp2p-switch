Intro

Installable via npm install --save libp2p-switch, it can also be used directly in the browser.

Download

The source is available for download from GitHub. Alternatively, you can install using npm:

$ npm install --save libp2p-switch

You can then require() libp2p-switch as normal:

const libp2PSwitch = require('libp2p-switch')

In the Browser

Libp2p-switch should work in any ES2015 environment out of the box.

Usage:

<script type="text/javascript" src="index.js"></script>

The portable versions of libp2p-switch, including index.js and index.min.js, are included in the /dist folder. Libp2p-switch can also be found on unpkg.com under

  • https://unpkg.com/libp2p-switch/dist/index.min.js
  • https://unpkg.com/libp2p-switch/dist/index.js

ConnectionFSM handles the complex logic of managing a connection between peers. ConnectionFSM is internally composed of a state machine to help improve the usability and debuggability of connections.
